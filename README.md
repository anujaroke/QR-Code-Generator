# QR Code Generator (Python)
A simple Python script to generate a QR code for any URL and save it using a file path defined in a `.env` file.

## Requirements
pip install qrcode[pil] python-dotenv

## Setup
Create a .env file in the project directory:
FILE_PATH=path/to/qr_code.png

## Run
python main.py
Enter a URL when prompted.
The QR code will be saved to the path specified in .env.

## Output
QR Code Generated Successfully
